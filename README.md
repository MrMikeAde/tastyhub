# 🍽️ TastyHub — Modern Restaurant Platform

TastyHub is a sleek, high-performance restaurant web application built with **Next.js**, designed to deliver an engaging food browsing experience and seamless online ordering.

---

## 🚀 Live Demo

👉 https://tastyhub-bwm.netlify.app/

---


## ✨ Features

- 🧾 Dynamic and visually appealing food menu
- 🛒 Smooth and intuitive ordering experience
- ⚡ Optimized performance with Next.js App Router
- 📱 Fully responsive across mobile, tablet, and desktop
- 🎨 Clean, modern UI/UX design
- 🔧 Scalable and maintainable code structure

---

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** JavaScript / TypeScript
- **Styling:** Tailwind CSS (update if different)
- **Runtime:** Node.js
- **Deployment:** Netlify

---

## 📦 Getting Started

### Prerequisites

- Node.js (LTS recommended)
- npm, yarn, or pnpm

---

### Installation

```bash
git clone <repository-url>
cd tastyhub-website

npm install
# or
yarn install
# or
pnpm install
🧪 Run Locally
npm run dev
# or
yarn dev
# or
pnpm dev

Open http://localhost:3000
 in your browser.
